Este projeto consiste em um sistema de gerenciamento de Pontos Turísticos, realizando cadastro, consulta, edição e remoção das informações.

**As tecnologias utilizadas foram:**

**Backend:** C#, ASP.NET Core, Entity Framework Core e SQL Server.

**Frontend:** React, Vite e Tailwind CSS.

O backend foi desenvolvido utilizando a arquitetura em camadas, separando as responsabilidades da aplicação para melhorar a organização, manutenção e escalabilidade.

**A estrutura do projeto segue o seguinte fluxo:**

Controller > Service > Repository > Database
